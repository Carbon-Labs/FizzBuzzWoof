# FizzBuzzWoof
A Fizz Buzz Woof implementation in Rust

# Compilation
Compile this by simply running
>cargo build

Then, run the program using
>cargo run

Of course, to save trouble, you *could* actually just do
>cargo run

Up to you to decide!
